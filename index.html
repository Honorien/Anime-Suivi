Bienvenue sur Suivi d'Anime !

Cette application vous permet de suivre facilement vos animes en cours et terminés.

Voici comment ça fonctionne :

  • Ajoutez un anime en remplissant le nom, la saison et l'épisode.
  • Utilisez les boutons pour marquer un anime comme terminé, le modifier ou le supprimer.
  • Recherchez et triez vos animes facilement.
  • Toutes vos données sont stockées localement dans le cache de votre navigateur.
  • Vous pouvez exporter votre liste pour la sauvegarder ou l'importer sur un autre appareil.
